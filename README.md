## Dev Setup

Clone or download repo and navigate to project folder.

Install dependencies
```
npm install
```

Start dev server
```
npm start
```

Load site at localhost:8080

## Build and Deploy

To build for production
```
npm run build
```

Bundle.js and index.html are generated in /public folder.
